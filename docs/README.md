# Documentación Jasboot (pública)

Este árbol contiene **solo textos explicativos**: referencia del lenguaje, especificaciones técnicas, estado del proyecto, contratos y guías. No incluye hojas de seguimiento con casillas, catálogos de tests ordenados por fases ni roadmaps tabulados; esos materiales viven **solo en el monorepo de desarrollo** (`docs/PROYECTO/`, `docs/TECNICO/SDK/` en el workspace Jasboot).
